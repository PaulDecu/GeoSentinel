Étape 1 : Créer le bundle manuellement Lance cette commande (à la racine du projet) pour packager le JS dans le dossier Android :


npx react-native bundle --platform android --dev false --entry-file index.js --bundle-output android/app/src/main/assets/index.android.bundle --assets-dest android/app/src/main/res


Étape 2 : Générer l'APK

Bash
cd android
./gradlew assembleDebug
L'APK généré dans android/app/build/outputs/apk/debug/app-debug.apk sera désormais autonome.

Parfait, voici la procédure standard pour mettre à jour votre dépôt GitHub avec vos nouvelles modifications.

Les commandes à taper
Tapez ces commandes successivement dans votre terminal, à la racine du projet :

Ajouter les modifications (les fichiers modifiés, ajoutés ou supprimés) :

Bash
git add .
Créer le commit avec un message descriptif :

Bash
git commit -m "Description de vos mises à jour"
Envoyer sur GitHub (le push) :

Bash
git push
